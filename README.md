# robot-list
a list of robots with cute avatars

## Features
- full list from robot api
- click avatar to see more information about the robot
- manage favorite list with the heart button

### Favorite List
- click heart to add/remove robots to favorite list
- show favorite list by clicking 'show favorite list' button
- click clear button to clear favorite list
- go back to home page to see full list of cute robots again!
